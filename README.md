## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/roscoejlr24/InkForge.git
   ```
2. Navigate to the project directory:
   ```bash
   cd InkForge
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
